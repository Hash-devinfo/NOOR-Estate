# Noor Estates 🏡

Noor Estates is a full-stack real estate web application built with the MERN stack 
(MongoDB, Express, React, Node.js). It connects property buyers and sellers on a 
single platform with real-time chat, inquiry management, and a powerful property 
search experience.

## Features
- 🔐 Role-based auth (Buyer / Seller / Admin)
- 🏘️ Property listings with image uploads via Cloudinary
- 🔍 Advanced search and filtering by city, type, price, and amenities
- 💬 Real-time chat between buyers and sellers (Socket.io)
- 📩 Inquiry system with read/unread tracking
- ❤️ Wishlist for saving favourite properties
- 📊 Seller dashboard with live stats and CSV export
- 📱 Fully responsive — mobile and desktop

## Tech Stack
- **Frontend:** React, React Router, Axios, Socket.io-client
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Auth:** JWT-based authentication
- **Storage:** Cloudinary for image uploads
- **Realtime:** Socket.io
